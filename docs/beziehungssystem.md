# 💑 Beziehungssystem – "Guild Life"

---

## 🌍 Grundkonzept

Beziehungen sind das soziale Netz deiner Dynastie. Jeder Charakter hat eine Meinung über dich (-100 bis +100). Heirat ist ein strategisches Werkzeug für Macht und Vermögen.

### Kernregeln

| Regel | Entscheidung |
|---|---|
| 👤 Geschlecht | Zufällig bei Geburt |
| 👶 Kinder | Unbegrenzt, ab 5. Kind sehr niedrige Chance |
| 🏳️‍🌈 Gleichgeschlechtlich | Nur als geheime Affäre (bei Entdeckung: Ketzerei) |
| 👑 Erben-Wahl | Frei wählbar zwischen allen Kindern |

---

## ❤️ Beziehungsstufen

| Stufe | Bereich | Bedeutung |
|---|---|---|
| 💀 **Erzfeind** | -100 bis -61 | Hass, aktive Feindschaft |
| 😡 **Rivale** | -60 bis -21 | Abneigung, Konkurrenz |
| 😐 **Neutral** | -20 bis +20 | Gleichgültig |
| 😊 **Freundlich** | +21 bis +60 | Sympathie, Zusammenarbeit |
| 🤝 **Verbündeter** | +61 bis +80 | Starkes Vertrauen |
| ❤️ **Vertraut** | +81 bis +100 | Tiefste Verbundenheit |

## 🧑‍🤝‍🧑 Beziehungstypen

### Familie

| Typ | Mechanik |
|---|---|
| 👨‍👩‍👧‍👦 **Ehepartner** | Wichtigste Beziehung, bringt Mitgift, beeinflusst Erben |
| 👶 **Kinder** | Erben der Dynastie, können erzogen werden |
| 👴 **Eltern** | Bestimmen Startbedingungen, können Erbe hinterlassen |
| 👫 **Geschwister** | Können Verbündete oder Rivalen sein |

### Sozial

| Typ | Mechanik |
|---|---|
| 🤝 **Freund** | Unterstützung bei Wahlen, Handel, Schutz |
| 😡 **Rivale** | Konkurrent in Wirtschaft und Politik |
| 💀 **Erzfeind** | Aktiver Gegner, plant gegen dich |
| 🧑‍🏫 **Mentor** | Erfahrener NPC, gibt Boni auf Ausbildung |
| 🤵 **Patron** | Reicher/mächtiger Gönner, unterstützt Aufstieg |

### Romantisch

| Typ | Mechanik |
|---|---|
| 💕 **Werben** | Vorstufe zur Heirat, Beziehung aufbauen |
| 💍 **Verlobt** | Heirat angekündigt, Mitgift wird verhandelt |
| 💑 **Verheiratet** | Feste Bindung, gemeinsame Kinder, Vermögenszusammenlegung |
| 💔 **Getrennt** | Nur durch Tod des Partners (keine Scheidung im Mittelalter!) |
| 🖤 **Affäre** | Geheim, riskant – bei Entdeckung: Skandal! |

## 💍 Heirat

### Heiratsvoraussetzungen

| Voraussetzung | Details |
|---|---|
| 💕 Beziehungswert | Mindestens +60 (Verbündeter) |
| 🎂 Alter | Beide mindestens 16 Jahre |
| ⛪ Glaube | Beide mindestens Glaube 30 (Kirche muss zustimmen) |
| 💰 Mitgift | Muss verhandelt werden |
| 👑 Stand | Beeinflusst, wer verfügbar ist |

### Heiratskandidaten nach Stand

| Dein Stand | Mögliche Partner |
|---|---|
| 🧑‍🌾 **Leibeigener** | Leibeigene, Handwerker (selten) |
| 🔨 **Handwerker** | Handwerker, Händler |
| 💼 **Händler** | Händler, Handwerker, Gelehrte |
| 📚 **Gelehrter** | Gelehrte, Händler, Patrizier (selten) |
| 🏰 **Patrizier** | Patrizier, reiche Händler, Gelehrte |

### Mitgift nach Stand

| Partner-Stand | Mitgift (Richtwert) | Zusätzlich |
|---|---|---|
| 🧑‍🌾 Leibeigener | 5–20 Gold | Arbeitskraft |
| 🔨 Handwerker | 30–80 Gold | Evtl. Werkzeuge/Materialien |
| 💼 Händler | 80–200 Gold | Handelskontakte |
| 📚 Gelehrter | 40–100 Gold | Intelligenz-Bonus für Erben |
| 🏰 Patrizier | 200–500 Gold | Ansehen-Bonus, politische Kontakte |

### Heirats-Ablauf

1. **Werben** → Beziehung aufbauen (mehrere Aktionen)
2. **Verlobung** → Mitgift verhandeln
3. **Hochzeit** → Kirchliche Trauung (Kosten: 30 Gold)

## 😡 Rivalitäten

### Wie entstehen Rivalitäten?

| Auslöser | Beziehungsänderung |
|---|---|
| 🏪 Gleicher Beruf in der Stadt | –5 pro Jahr (Konkurrenz) |
| 🗳️ Wahlniederlage gegen jemanden | –15 |
| 🔧 Sabotage gegen jemanden | –30 |
| 📜 Anklage gegen jemanden | –25 |
| 💀 Mord an Familienmitglied | –80 (sofort Erzfeind) |
| 💍 Heiratsrivalität | –20 |
| 💰 Geschäftlicher Betrug | –25 |
| 📜 Verleumdung | –15 |

### Was können Rivalen tun?

| Rivale-Stufe | Mögliche Aktionen des Rivalen |
|---|---|
| 😡 **Rivale** (-60 bis -21) | Höhere Preise im Handel, politische Gegnerschaft, Gerüchte |
| 💀 **Erzfeind** (-100 bis -61) | Sabotage, Anklagen, Attentatsversuche, Handelsblockaden |

### Rivalitäten beenden

| Aktion | Effekt | Kosten |
|---|---|---|
| 🤝 **Friedensangebot** | +20 Beziehung (wenn akzeptiert) | Charisma-Check |
| 🎁 **Großzügiges Geschenk** | +10–25 Beziehung | 50–200 Gold |
| 💍 **Heirats-Allianz** | +40 Beziehung, Familien verbunden | Mitgift + Beziehung 0+ nötig |
| 💀 **Rivalen eliminieren** | Problem gelöst, aber... | Hoher Verdacht, Erzfeindschaft mit Familie |
| ⚖️ **Gerichtlicher Vergleich** | +15 Beziehung | 25 Gold (Jurist) |
