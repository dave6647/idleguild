# 📋 Charaktersystem – "Guild Life"

---

## 🧬 Grundattribute

5 Attribute, bei Geburt **komplett zufällig** vergeben (0–100), kein Re-Roll.

| Attribut | Symbol | Beschreibung |
|---|---|---|
| **Stärke** | 💪 | Körperliche Kraft |
| **Geschick** | 🎯 | Fingerfertigkeit & Präzision |
| **Charisma** | 🗣️ | Überzeugungskraft & Ausstrahlung |
| **Intelligenz** | 🧠 | Wissen & Lernfähigkeit |
| **Frömmigkeit** | 🙏 | Gottesfürchtigkeit & Moral |

---

## 📈 Sekundärwerte (dynamisch)

| Wert | Symbol | Mechanik |
|---|---|---|
| **Gesundheit** | ❤️ | Sinkt durch Alter, Krankheit, Kampf. Bei 0 → Tod |
| **Ansehen** | ⭐ | Steigt/sinkt durch Taten & Stand. Bestimmt sozialen Zugang |
| **Reichtum** | 💰 | Gold, Betriebe, Besitz. Ermöglicht Aktionen & Aufstieg |
| **Verdacht** | 👁️ | Steigt mit Straftaten, sinkt langsam (–3/Jahr). Zu hoch → Verfolgung & Kerker |
| **Glaube** | ✝️ | Bestimmt durch Spenden & Verhältnis zu Kirchenwürdenträgern. Zu niedrig → Ketzerei |

---

## 👁️ Verdacht-System

Verdacht steigt dynamisch mit jeder begangenen Straftat und sinkt nur langsam über die Zeit.

### Verdacht-Quellen

| Aktion | Verdacht-Anstieg |
|---|---|
| 🔧 Sabotage (erfolgreich) | +10 |
| 🔧 Sabotage (fehlgeschlagen) | +25 |
| 💰 Bestechung (erfolgreich) | +5 |
| 💰 Bestechung (fehlgeschlagen) | +20 |
| 🗡️ Intrigen (fehlgeschlagen) | +15 |
| 🧪 Giftmord (erfolgreich) | +20 |
| 🧪 Giftmord (fehlgeschlagen) | +40 |
| ⚔️ Einschüchterung (fehlgeschlagen) | +15 |
| 🏴 Diebstahl | +10–30 |

### Verdacht-Senkung

| Aktion | Verdacht-Senkung |
|---|---|
| ⏳ Pro Jahr (natürlich) | –3 |
| ✝️ Spende an die Kirche | –5 |
| 📜 Freispruch vor Gericht | –15 |
| 🤝 Gute Taten / Wohltätigkeit | –5 |
| 💰 Bestechung der Wache (riskant!) | –10, aber +5 bei Fehlschlag |

### Verdacht-Stufen & Konsequenzen

| Stufe | Bereich | Konsequenzen |
|---|---|---|
| ✅ Unauffällig | 0–20 | Keine Auswirkungen |
| ⚠️ Gerüchte | 21–40 | Misstrauen, höhere Preise, politische Nachteile |
| 🔶 Beobachtung | 41–60 | Stadtwachen, Durchsuchungen, –10% auf Straftaten |
| 🔴 Verfolgung | 61–80 | Ermittlungen, Verhaftungen möglich, Ämter gesperrt |
| 💀 Gejagt | 81–100 | Haftbefehl, Beschlagnahmung, Kerker oder Hinrichtung |

---

## ✝️ Glaube-System

Glaube wird nicht durch innere Überzeugung bestimmt, sondern durch das konkrete Verhältnis zu Kirchenwürdeträgern und durch Spenden an die Kirche.

### Glaube-Quellen

| Aktion | Glaube-Änderung |
|---|---|
| 💰 Spende an die Kirche (klein) | +3 |
| 💰 Spende an die Kirche (groß) | +8 |
| 💰 Kirchenbau finanzieren | +15 |
| 🤝 Freundschaft mit Priester/Bischof | +5 pro Jahr |
| ⛪ Regelmäßiger Kirchgang | +2 pro Jahr |
| 🎉 Kirchliches Fest ausrichten | +10 |
| 😤 Streit mit Kirchenwürdenträger | –10 |
| 🧪 Giftmord / schwere Sünde | –15 |
| 📜 Als Ketzer beschuldigt | –20 |
| ❌ Kirchgang verweigern | –3 pro Jahr |
| 💀 Exkommunikation | → Glaube auf 0 |

### Glaube-Stufen & Auswirkungen

| Stufe | Bereich | Auswirkungen |
|---|---|---|
| 💀 Ketzer | 0–15 | Exkommunikation, Bürger meiden dich, Inquisition |
| ⚠️ Sünder | 16–35 | Kein Zugang zu Segnungen, politische Nachteile |
| 😐 Gläubig | 36–60 | Normaler Status, Zugang zu Kirchgang |
| 😇 Fromm | 61–80 | Kirchliche Unterstützung, Segnungen, Verdacht sinkt schneller |
| 🌟 Auserwählt | 81–100 | Bischof als Verbündeter, Schutz vor Anklage, Kirchenämter |

## 🐍 Abgeleitete Aktionen

Es gibt kein eigenes Attribut für List. Stattdessen berechnen sich dunkle Aktionen aus Kombinationen der Grundattribute.

| Aktion | Berechnung | Schwelle | 50%-Wert |
|---|---|---|---|
| 📜 **Anklage** | (Intelligenz + Charisma) / 2 | 30 | 50 |
| 💰 **Bestechung** | (Charisma + Intelligenz) / 2 | 35 | 55 |
| 🗡️ **Intrigen** | (Charisma + Geschick) / 2 | 40 | 60 |
| 🔧 **Sabotage** | Geschick | 45 | 65 |
| ⚔️ **Einschüchterung** | (Stärke + Charisma) / 2 | 50 | 70 |
| 🧪 **Giftmord** | (Geschick + Intelligenz) / 2 | 55 | 75 |

### Erfolgswahrscheinlichkeit

Die Erfolgswahrscheinlichkeit skaliert von ca. 15% an der Schwelle bis maximal 90% bei einem Wert von 100. Eine hundertprozentige Erfolgschance gibt es nie – es bleibt immer ein Restrisiko.

| Wert | Ungefähre Erfolgswahrscheinlichkeit |
|---|---|
| Bei Schwelle | ~15% |
| Schwelle + 10 | ~33% |
| Bei 50%-Wert | ~50% |
| 50%-Wert + 15 | ~72% |
| Bei 100 | ~90% |

## 🏷️ Traits

Jeder Charakter erhält bei der Geburt zufällig **1 Trait**. Im Laufe des Lebens können durch Entscheidungen und Events bis zu **2 weitere** hinzukommen. Es sind **maximal 3 Traits** gleichzeitig aktiv. Erhält ein Charakter einen neuen Trait bei bereits 3 aktiven, muss ein bestehender abgelegt werden.

### Angeborene Traits (Beispiele)

| Trait | Effekt |
|---|---|
| 🌟 *Wunderkind* | +15 Intelligenz |
| 💪 *Bärenstark* | +15 Stärke |
| 🤒 *Kränklich* | Gesundheit sinkt schneller |
| 🍀 *Glückskind* | Bessere Zufallsevents |
| 👿 *Jähzornig* | Öfter Konflikte, +Einschüchterung |
| 😇 *Gütig* | +Ansehen, schwerer zu intrigieren |

### Erworbene Traits (Beispiele)

| Trait | Wie erworben | Effekt |
|---|---|---|
| 🗡️ *Duellant* | 3 Duelle gewonnen | +Kampf, +Ansehen |
| 🐀 *Giftmischer* | Jemanden vergiftet | +Giftmord-Chance, +Verdacht |
| 📜 *Gelehrt* | Universität abgeschlossen | +Intelligenz, Zugang zu Ämtern |
| 🍺 *Trinker* | Zu oft in der Taverne | –Gesundheit, +Charisma (manchmal) |
| 🤝 *Meister der Gilden* | Gildenmeister geworden | +Handwerk, +Ansehen |
| ⛪ *Ketzer* | Gegen die Kirche gehandelt | –Glaube, +Verdacht |

## 👶 Geburt & Stände

Der Stand der Familie beeinflusst die Startbedingungen des Charakters.

| Stand | Startgold | Attribut-Bonus | Zugang |
|---|---|---|---|
| 🧑‍🌾 **Leibeigene** | Sehr wenig | +Stärke | Harte Arbeit, schwieriger Aufstieg |
| 🔨 **Handwerker** | Wenig | +Geschick | Werkstätten, Gilden |
| 💼 **Händler** | Mittel | +Charisma | Märkte, Handelsrouten |
| 📚 **Gelehrte** | Wenig–Mittel | +Intelligenz | Universität, Kirche |
| 🏰 **Patrizier** | Viel | +Charisma | Politik, hohe Ämter |

## 🎂 Lebensphasen

| Phase | Alter | Gameplay |
|---|---|---|
| 👒 **Kindheit** | 0–6 | Nur Events, Charakter wird geformt |
| 📖 **Jugend** | 7–14 | Ausbildung wählen, erste Entscheidungen |
| ⚒️ **Lehrjahre** | 15–20 | Beruf erlernen, erste Arbeit |
| 🏛️ **Erwachsenenalter** | 21–50 | Volles Gameplay: Beruf, Politik, Familie |
| 👴 **Alter** | 51+ | Stats sinken, Erbe vorbereiten |

## 🧬 Vererbung (Dynastie)

Wenn ein Charakter stirbt, geht das Spiel mit dem Erben weiter.

| Element | Mechanik |
|---|---|
| **Attribute** | (Elternteil 1 + Elternteil 2) / 2 ± Zufallsvariation (±10) + Stand-Bonus |
| **Familienname** | Bleibt erhalten |
| **Vermögen & Betriebe** | Werden vererbt (abzüglich Erbsteuer) |
| **Ansehen** | Teilweise übernommen |
| **Traits** | 30% Vererbungschance |
| **Rivalitäten** | Bleiben bestehen |
